構造体は、メンバ変数として「自分と同じ型のポインタ」を持つことができる。例えば次の通りである。

リスト構造

教科書「やさしいC」の「11.4 構造体の応用」の「構造体でリストを作る」を読んで、リスト構造のしくみについて確認しよう。
自己参照構造体は、2 年次で本格的に学ぶ「データ構造とアルゴリズム」で非常に重要な働きをする。 「データ構造とアルゴリズム」は効率のよいプログラムを書くために欠かせない。 今回は基本的なデータ構造として、「リスト構造」を扱う。
一般にデータ構造とは、特定の操作を行いやすくするために、規則に基づいて設計されたデータの形式である。複雑なプログラムを作るときの部品あるいは骨組みとなる。 配列もデータ構造のひとつである。では、今回扱うリスト構造は何の役に立つのかを説明する。 例えば、配列のようにデータが並んでいるとき、その途中に別の要素を挿入したいとする。 配列の場合は、すでに要素数ぴったりのサイズであれば、もう要素を増やすことはできない。より大きな配列を用意し、途中まで元の配列の要素をコピーして、新しい要素を挿入し、その後に残りの要素をコピーすることになる。このような操作は、配列の要素数が少ない場合や、要素ひとつあたりのデータサイズが小さい場合は大したことはないが、配列の要素数が増えたり、要素ひとつあたりのデータサイズが大きくなったりすると時間がかかるようになる。

配列に要素を追加する様子.
一方、リスト構造はポインタを用いてデータを数珠つなぎにしたもので、途中に要素を追加しても、余計なコピーなどの操作は発生しない。要素の挿入だけでなく、途中の要素を削除する必要があるときも、リスト構造は効率的である。

リスト構造に要素を追加する様子.
前述のとおり、データ構造は、特定の規則に基づいて設計される。リスト構造の場合を説明する。まず、数珠つなぎにしたいデータを自己参照構造体として表現する。
?

typedef struct _node {
    int value;      /* 保持するデータ。この場合は整数値 */
    struct _node *next; /* 数珠つなぎにするためのポインタ */
} Node;
そして、リスト構造の先頭と末尾へのポインタを保持した構造体を作り、これをリスト構造とする。今回扱うリスト構造は特に「連結リスト (Linked List)」(より詳しくは「片方向リスト」)という。
?
1
2
3
4
typedef struct {
    Node *head; /* 先頭へのポインタ */
    Node *tail; /* 末尾へのポインタ */
} LinkedList;
ここで、リスト構造でいう「特定の規則」とは、次のようなものである。
  • リスト構造に要素が何もない場合、先頭へのポインタ head と末尾へのポインタ tail はともに NULL とする。
  • リスト構造の各要素が持つポインタ next は、数珠つなぎになった次の要素を指す。
  • 末尾の要素の持つポインタ next は NULL とする。
リスト構造への操作は、これらの規則を順守しつつ行う。

    typedef struct {
        Node *head;
        Node *tail;
    } LinkedList;
     
    /* リストの要素ひとつ分のメモリを確保し、メンバ変数の値を代入 */
    Node *CreateNode(int no, const char *name)
    {
        Node *n = (Node *)malloc(sizeof(Node));
     
        if (n == NULL)
        {
            fprintf(stderr, "CreateNode: error: memory allocation failed\n");
            return NULL;
        }
     
        n->data.no = no;
        strcpy(n->data.name, name);
     
        return n;
    }
     
    /* リストの要素の表示 */
    void PrintNode(Node *node)
    {
        printf("no: %d, name: %s\n", node->data.no, node->data.name);
    }
     
    /* リストを初期化 */
    void ListInit(LinkedList *list)
    {
        list->head = NULL;   /* リストの先頭を NULL に */
        list->tail = NULL;   /* リストの末尾を NULL に */
    }
     
    /* リストの末尾に要素を追加 */
    void ListAddTail(LinkedList *list, Node *node)
    {
        if (list->head == NULL) /* 要素がまだないとき */
        {
            /* 要素をひとつ追加すると、それが先頭かつ末尾になる */
            list->head = node;
            node->next = NULL;
            list->tail = node;
        }
        else
        {
            list->tail->next = node;
            node->next = NULL;
            list->tail = node;
        }
    }
     
    /* リストの先頭を削除 */
    void ListDeleteHead(LinkedList *list)
    {
        if (list->head == NULL)
        {
            /* 要素がないなら何もする必要がない */
            return;
        }
        else
        {
            Node *next = list->head->next; /* 先頭の次の要素へのアドレスを保持 */
     
            /* 先頭を削除…削除の前に次の要素へのアドレスを記憶しておかないと削除したら失われる */
            free(list->head);
     
            /* 新しい先頭要素として、元々の先頭の次の要素を設定 */
            list->head = next;
     
            /* 先頭を削除して要素がなくなった場合、末尾も NULL にする */
            if (list->head == NULL)
                list->tail = NULL;
        }
    }
     
    /* リストを表示 */
    void ListPrint(LinkedList *list)
    {
        if (list->head == NULL)
        {
            puts("(null)\n");
        }
        else
        {
            Node *p = list->head;
     
            while ( p != NULL )
            {
                PrintNode(p);
                p = p->next;
            }
            putchar('\n');
        }
    }
     
    /* リストの要素を削除 */
    void ListClear(LinkedList *list)
    {
        Node *p = list->head;
     
        while ( p != NULL )
        {
            Node *tmp = p;  /* 現在のノードへのポインタを保存しておく */
            p = p->next; /* p を先に進める…進める前に p を解放するとリストを辿るためのポインタも失われる */
            free(tmp);      /* 現在のノードのメモリを解放する */
        }
     
        /* リストを初期化して、次の利用に備える */
        ListInit(list);
    }
     
    int main(void)
    {
        LinkedList list;
     
        ListInit(&list);
     
        ListAddTail(&list, CreateNode(100, "Yamada"));
        ListAddTail(&list, CreateNode(200, "Suzuki"));
        ListAddTail(&list, CreateNode(300, "Sato"));
     
        ListPrint(&list);
     
        ListDeleteHead(&list);
     
        puts("head deleted: ");
     
        ListPrint(&list);
     
        ListClear(&list);
     
        return 0;
    }
