関数ポインタ

教科書「やさしいC」の「10.5 関数ポインタ」を読んで、関数ポインタのしくみについて確認しよう。
変数に対してポインタがあるように、関数にもポインタがある。関数はプログラムの実行時にメモリに格納される。関数ポインタは、関数がメモリ上のどこに格納されているか、というアドレスを保持する。変数のポインタでは、int 型、char 型、という風に、変数の型ごとにポインタを宣言した。関数ポインタでも、関数の「型」ごとに宣言する。ここでいう関数の「型」とは、関数の返り値と引数の型の組み合わせである。関数ポインタは、同じ「型」の関数のアドレスを保持することができる。
例えば、返り値が int で、引数として int n を取るような関数のポインタ p は、次のように宣言できる。 
int (*p)(int n);
関数ポインタ p を間接参照し、関数として使うときには、変数のポインタと同様に * をつけて、 
(*p)( 3 );
などと呼び出すこともできるが、いちいち * をつけるのも面倒なので、* をつけずに 
p( 3 );
と呼び出すことも可能になっている。

    関数ポインタの重要な使い方として「関数 f の引数として別の関数 g (のポインタ) を渡し、関数 f の内部で g を呼び出して使う」というものがある。 関数 f の内部で呼び出される関数 g のことを、コールバック関数と呼ぶ。コールバック関数を使うと、似たような処理だがちょっと違うという場合 (例えば扱うデータの型が違う、条件分岐の条件が違う) に、違いをコールバック関数で吸収でき、プログラムのコード数を減らすことができる。コールバック関数は、グラフィカルユーザインタフェース (GUI) において、「このボタンを押したら◯◯という関数を呼び出す」といった処理で一般的に用いられている。
    ここでは、コールバック関数を使う例として、C 言語の標準ライブラリに含まれている qsort 関数を扱う。 qsort 関数は、並べ替え (ソーティング; sorting) を行う関数である (ソーティングのアルゴリズムは様々あるが、高速なソーティングのうちのひとつであるクイックソート (quick sort) を行う関数である。ソーティングについて詳しくは 2 年次の「データ構造とアルゴリズム」で学ぶ)。 これまでは、扱うデータの型ごとにソーティングを行う処理を書いてきたが、それは面倒である。 qsort 関数は、任意の型のデータを扱えるような汎用的な仕組みを実現するために、コールバック関数を用いている。
    qsort 関数のプロトタイプ宣言は stdlib.h に書かれており、次のようになっている。 
    void qsort(void *base, size_t num, size_t size, int (*compare)(const void* a, const void* b));
    ここで、base は配列の先頭アドレス、num は配列の要素数、size は配列の要素ひとつあたりのバイト数、compare はデータを比較するためのコールバック関数である。 ここで使われている void 型のポインタは、任意の型のアドレスを受け取ることができる。すると、実際に base に配列が与えられたとき、型がわからないので、その配列の要素を順にアクセスするのに何バイトずつ移動したらよいかわからない。そこで size で要素ひとつあたりのバイト数を指定する。
    compare 関数に渡される引数は、配列の要素のアドレスである。例えば、int 型の配列が入力なら、compare 関数の引数には int * が渡される。引数の値を参照するには、実際の型にキャストしたのち (例えば (int *)a)、関節参照する (例えば *(int *)a)。なお compare 関数は、次のような仕様で作らなければならない。
    • もし *a < *b であれば、返り値は負の値
    • もし *a == *b であれば、返り値は 0
    • もし *a > *b であれば、返り値は正の値

    int main(void)
    {
        int i;

        printf("descending order:\n");
        print_array(N, A);
        putchar('\n');

        print_array(N, A);
     
        return 0;
    }
     
     /* 降順、つまり徐々に値が小さくなるように並べ替えるための比較関数 */
    int descending_cmp(const void *a, const void *b)
    {
        /* 実際のデータ型である int にキャスト */
        int aInt = *(int *)a;
        int bInt = *(int *)b;
     
        /* bInt < aInt なら負、bInt == aInt なら 0、bInt > aInt なら正 */
        return bInt - aInt;
    }
     
     /* 昇順、つまり徐々に値が大きくなるように並べ替えるための比較関数 */
    int ascending_cmp(const void *a, const void *b)
    {
        /* 実際のデータ型である int にキャスト */
        int aInt = *(int *)a;
        int bInt = *(int *)b;
     
        /* bInt > aInt なら負、bInt == aInt なら 0、bInt < aInt なら正 */
        return aInt - bInt;
    }

    {
        int i;
        for (i=0; i<n; i++)
            printf("a[%d] = %d\n", i, a[i]);
    }
